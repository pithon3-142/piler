# Piler

This is a simple python program created to package desktop files into order. 
This app is inspired by my junk files lying around everywhere without order.

# Usage :

# Step 1 :
Select the Source Directory from which to move the files from

# Step 2 :
Select the category of the File Types you would like to sort

# Step 3 :
Select the file type to sort

# Step 4 :
Select the Target Directory (Destination) for which to move the files

# Step 5 :
Click the "Organize" button and let the program tidy up your computer.
